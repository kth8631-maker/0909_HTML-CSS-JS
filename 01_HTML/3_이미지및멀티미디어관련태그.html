이미지 관련 태그

src속성, alt속성

* alt
- 사진의 경로가 존재하지 않거나 표현할 수 없는 경우 대체 텍스트
- 스ㅋ린리더(화면 낭독기)에서 이미지를 읽어주는 설명문구
웹접근성 => 사용자의 신체적, 지역, 나이, 지식수준 등을 고려하여 가능한 최대한 많은 사용자가 불편함 없이 사용할 수 있도록 웹을 만드는 것

* width와 height 속성

요소의 가로, 세로길이를 조정할 수 있는 속성 고정길이(px) 가변길이(%)로 지정이 가능하다

고정길이 단위(px) == 기본값
